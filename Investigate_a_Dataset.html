Introduction¶

This dataset collects information from 100k medical appointments in Brazil and is focused on the question of whether or not patients show up for their appointment. A number of characteristics about the patient are included in each row.

‘ScheduledDay’ tells us on what day the patient set up their appointment. ‘Neighborhood’ indicates the location of the hospital. ‘Scholarship’ indicates whether or not the patient is enrolled in Brasilian welfare program Bolsa Família. Be careful about the encoding of the last column: it says ‘No’ if the patient showed up to their appointment, and ‘Yes’
if they did not show up.

#Method to calculate probability for non-categorical data i.e 'age' 
def calc_probability(column):
    prob_df = pd.crosstab(index=no_show[column], columns=no_show['no_show']).reset_index()
    prob_df['prob_show_up'] = prob_df['No'] / (prob_df['No'] + prob_df['Yes'])
    return( prob_df[[column, 'prob_show_up']])

# Method for calculating the probability for categorical data 
def calc_prob_category(column):
    rows = []
    for value in no_show[column].unique():
        total = len(no_show[no_show[column] == value])
        show_up_count = len(no_show[(no_show[column] == value) & (no_show['no_show'] == 'No')])
        if total != 0:
            probability = show_up_count / total
        row={column: value, 'Probability': probability}
        rows.append(row)
    return pd.DataFrame(rows)

#Method for Drawing the Bar Plot 
def draw_barplot(column):
    sns.set_style("whitegrid")
    fig, ax = plt.subplots(figsize=(8,5))
    sns.barplot(data = calc_prob_category(column), x=column, y='Probability',ax=ax )
    plt.title('Probability of Showing Up with respect to '+ column.replace('_' , ' '))
    plt.ylabel('Probality of Showing Up')
    plt.xlabel(column.upper().replace('_' , ' '));
